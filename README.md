# 工具安装文档

记录各工具在不同环境下的安装方法、踩坑记录和对应版本号，方便以后快速复现。

## 使用方式

```bash
pip install mkdocs-material
```

本地预览：

```bash
mkdocs serve
```

打开 `http://localhost:8000` 即可浏览。
