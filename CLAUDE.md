# CLAUDE.md — 文档仓库

基于 **MkDocs + Material 主题**的个人技术文档仓库。

项目路径：`/home/ailab/tools_docs`

## 核心原则

**每篇文档的目标是让读者完整复现。** 按步骤操作就得到相同结果，不需要自行推断。

## 编写规范

**严格遵循 `docs/guide.md`。** 核心约束：

1. 版本精确到小版本号，用版本表列出所有组件
2. 步骤从零到验证，覆盖 前置→安装→配置→启动→验证
3. 命令直接可复制，不含占位符
4. 「注意」是事前提醒，「踩坑」是事后记录
5. 现象给完整报错信息

## 添加文档

1. 新文档 → `docs/<分类>/<文件名>.md`
2. 追加到已有分类 → 在 `index.md` 追加 `##` 段落
3. 新增分类 → 创建目录 + 更新 `mkdocs.yml` 的 `nav`

## 部署

```bash
mkdocs serve     # 本地预览 http://localhost:8000
mkdocs gh-deploy # 发布
```
